# 코드 리뷰

## 실행 순서

### 1. 변경 사항 확인
```bash
git diff --name-only
git diff
```

### 2. 리뷰 체크리스트

#### TypeScript (Frontend)
- [ ] `any` 타입 사용 없음
- [ ] Props 인터페이스 정의됨
- [ ] 에러 핸들링 있음
- [ ] 로딩 상태 처리
- [ ] 'use client' 적절히 사용

#### Python (Backend)
- [ ] 타입 힌트 있음
- [ ] async/await 올바르게 사용
- [ ] 에러 핸들링 (try-except)
- [ ] Pydantic 모델 사용
- [ ] 보안 (인증, 권한) 확인

#### 공통
- [ ] 하드코딩된 값 없음
- [ ] 환경 변수 사용
- [ ] 주석/문서화 적절함
- [ ] 네이밍 컨벤션 준수

### 3. 보안 체크
- [ ] API 키 노출 없음
- [ ] SQL 인젝션 방지
- [ ] XSS 방지
- [ ] 인증/인가 확인

### 4. 성능 체크
- [ ] N+1 쿼리 없음
- [ ] 불필요한 리렌더링 없음
- [ ] 적절한 캐싱

## 출력 형식

```
## 리뷰 결과

### ✅ 잘된 점
- ...

### ⚠️ 개선 필요
- 파일: 라인 - 내용

### 🔴 반드시 수정
- 파일: 라인 - 내용

### 💡 제안
- ...
```
